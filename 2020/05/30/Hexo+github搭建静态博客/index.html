Hexo+github搭建静态博客

下载并安装node.js

#Mac下使用brew安装
ben@benのMBP ~ : brew install node.js

#查看版本信息
ben@benのMBP ~ : node -v
ben@benのMBP ~ : npm -v

本地hexo部署

  • 官方文档：[https://hexo.io/zh-cn/docs/]
ben@benのMBP ~ : cd opt/blog #创建博客目录
ben@benのMBP blog : npm install -g hexo-cli #安装Hexo客户端
ben@benのMBP blog : hexo init benbenit #初始化博客目录
ben@benのMBP blog : cd benbenit && npm install #进入博客目录

本地测试

ben@benのMBP benbenit : hexo clean && hexo g && hexo s #清理缓存 && 生成静态文件 && 启动服务器

http://localhost:4000/ #浏览器访问本地部署的hexo

本地密钥配置

ben@benのMBP benbenit : ssh-keygen -t rsa -b 4096 -C "893880071@qq.com"

Github配置

#注册 用户名 
benbenit
#创建仓库，切记仓库前缀和用户名一致 
benbenit.github.io 

#将本地公钥复制到git 
ben@benのMBP benbenit : subl /Users/ben/.ssh/id_rsa.pub 

#测试连接 
ben@benのMBP benbenit : ssh -T git@github.com 

#出现以下文字说明成功 
Warning: Permanently added the RSA host key for IP address 'xxx.xxx.xxx.xxx' to the list of known hosts. Hi benbenit! You've successfully authenticated, but GitHub does not provide bash access. 

#配置本地git信息 
ben@benのMBP benbenit : git config --global user.name "benbenit" 
ben@benのMBP benbenit : git config --global user.email "893880071@qq.com"

部署Hexo到Github

#安装客户端 
ben@benのMBP benbenit : npm install hexo-deployer-git --save 

#修改benbenit下的_config.yml 
deploy: 
  type: git
  repo: git@github.com:benbenit/benbenit.github.io.git
  branch: master 
  
#部署 
ben@benのMBP benbenit : hexo clean && hexo g && hexo deploy 

#测试 
浏览器打开: benbenit.github.io

    恢复

背景

  • 命令行不小心rm -rf掉了，原始的md文件丢掉，只剩下git上面html文件

对策1

  • 百度html转markdown，转过来再自己一点一点排版，或者自己写代码一点点解析

对策2

  • 虽然这也是一种手动修改的方式，但效率比第一种高了太多

  • 使用Safari浏览器打开自己的文章，比如这篇C语言基础

  • 进入阅读模式(或者使用谷歌浏览器插件ReadView等进入阅读模式) | command + a | command + c
  • 打开Typora | command + v | 修改代码块语言

Typora图片问题

  • 在source文件夹下建立images文件夹
ben@bendeMacBook Pro benbenit % cd source && mkdir images
  • 对Typora设置
  • 每次写文章的时候YAML Front Matter里面添加
typora-root-url: ../
