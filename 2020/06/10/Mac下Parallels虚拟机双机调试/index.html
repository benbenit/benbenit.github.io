2020-06-10- 2020-08-15
调试-Windbg

Mac下Parallels虚拟机双机调试

系统

  • Mac + Parallels Desktop
  • MSDN，我告诉你。下载WinXP-SP3

使用Parallels安装两台xp系统

  • 被调试机器，WinXP

    • 添加串行口

  • 调试机器，WinXPServer

被调试机器(WinXP)设置

  • boot.ini是系统隐藏文件，设置成可见
  • 打开boot.ini
[boot loader]
timeout=30
default=multi(0)disk(0)rdisk(0)partition(1)\WINDOWS
[operating systems]
multi(0)disk(0)rdisk(0)partition(1)\WINDOWS="Microsoft Windows XP Professional[Debug]" /noexecute=option fastdetect /debug /debugport=com1
multi(0)disk(0)rdisk(0)partition(1)\WINDOWS="Microsoft Windows XP Professional"

调试机器(WinXPServer)设置

  • 安装windbg
  • 安装WinxpSP3符号表
  • 亦可自行搜索winxp的符号表
  • 安装时候有路径选项，默认是C:\WINDOWS\Symbols，我把他安装在C:\Symbols
  • 打开windbg -> File -> Symbol File Path，把上一步设置的安装路径写进去
  • 发送快捷方式到桌面，或者直接创建快捷方式，目的是为了修改启动参数
  • 右键快捷方式，选择属性，修改目标，添加参数

开始双机调试

调试机器(WinXPServer)

  • 双击快捷方式，启动windbg
  • file - kernel debug - com
  • 点击确定，等待被调试机器

被调试机器(WinXP)

  • 重启被调试机器(WinXP)
  • 选择开启调试，释放光标之后就可以在Windbg里面进行调试了
