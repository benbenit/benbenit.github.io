方案二

  • 因为Mac已经走代理，设置局域网走代理

  • vmwarefunction设置

  • v2ray设置，把监听本机IP改成监听局域网IP
  • ubuntu设置
  • 修改后重启ubuntu网络，修改.bashrc
export ALL_PROXY=socks5://0.0.0.0:1080
  • 这种方式是最好的，下jdk源码一次错误都没出

缩放

#在Wayland上
gsettings set org.gnome.mutter experimental-features "['scale-monitor-framebuffer']"

#在Xorg上运行
gsettings set org.gnome.mutter experimental-features "['x11-randr-fractional-scaling']"

#查看
ben@ben:~$ echo $XDG_SESSION_TYPE
x11

#所以运行上面第二条命令

其他安装

#vim
sudo apt install -y vim
#curl
sudo apt install -y curl
#wget
sudo apt install -y wget
#git
sudo apt install git

#在ubuntu里没有chkconfig,但有界面操作sysv-rc-conf,作用一样
vim /etc/apt/sources.list
deb http://archive.ubuntu.com/ubuntu/ trusty main universe restricted multiverse
sudo apt update
sudo apt install -y sysv-rc-conf
sudo cp /usr/sbin/sysv-rc-conf /usr/sbin/chkconfig

#方便使用netstat
sudo apt install -y net-tools
