MacBook初始化

  • 之前自己误操作把Docement都删掉了，索性重装系统把用到的记录下来，方便下次rm -rf

开启root用户

  • 系统偏好设置 | 用户与群组 | 登录选项 | 网络账户服务器 - 加入 | 打开目录实用工具 | Active Directory | 菜单栏 - 编辑 | 启用root用户

修改用户名

ben@bendeMacBook Pro ~ % sudo scutil --set HostName Ben

开启任何来源

ben@benのMBP ~ % sudo spctl --master-disable

安装Homebrew

域名映射

  • 因为Git的DNS污染，直接安装homebrew可能装不上，找个域名解析的网站，解析raw.githubusercontent.com和github.com。把解析出来的ip写到/etc/hosts里面
ben@benのMBP ~ % sudo vim /etc/hosts

安装Homebrew

ben@benのMBP ~ % ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

关闭Homebrew自动更新

ben@benのMBP ~ % echo 'export HOMEBREW_NO_AUTO_UPDATE=true' >> ~/.zshrc

  • 再次使用brew安装
ben@benのMBP ~ % brew cask install homebrew/cask-versions/java8
  • 导致的小问题，就是使用brew升级的时候会报错
  • 解决办法
~$ mkdir /usr/local/Caskroom/java8/.metadata/8u261,b12/a4634525489241b9a9e1aa73d9e118e6/Casks/       
~$ cp /usr/local/Caskroom/java8/.metadata/8u261,b12/a4634525489241b9a9e1aa73d9e118e6/jdk-8u261-macosx-x64.dmg/20200811085536.425/Casks/java8.rb /usr/local/Caskroom/java8/.metadata/8u261,b12/a4634525489241b9a9e1aa73d9e118e6/Casks/java8.rb
  • 这样虽然最后仍然会报错，但是检查到java8.rb文件存在了，就不会导致其他软件也更新不了。而java8本来就没想更新，这个问题也就解决了
  • 上面这种修改rb文件的方式也适合于安装其他本地dmg，软件升级也同样适用。比如升级百度云(绿色马赛克)失败

命令行命令

find

  • 查找符合条件的文件
参数 含义
-amin 查找指定时间内被存取过的文件或目录，单位分钟
-cmin 查找指令时间内被更改过的文件或目录，单位分钟
-exec 如果找到这个文件，即find指令的回传值为true，就执行该指令
-fstype<文件系统类型> 寻找特定文件系统类型的文件或目录
-type<文件类型> 寻找符合指定的文件类型的文件，f普通文件，d目录，l符号链接
-uid<用户id> 寻找属于指定用户id的文件
-user<拥有者名称> 寻找属于指定用户的文件
-iname 查找的时候忽略大小写
-regex 基于正则表达式匹配
-delete 查找出来，删除，比如find ./ -name f “*.txt” -delete ,删除当前目录所有txt文件
-perm 根据文件所有权进行匹配，find . -type f -name “*.java” ! -perm 777,找出当前文件夹下权限不是777的所有java文件
  • 借助-exec选项与其他命令结合使用，但是-exec不能执行多行命令，可以把命令封装成sh文件，然后接在-exec上面
#将30天前的.log文件移动到old目录中
find . -type f -mtime +30 -name "*.log" -exec cp {} old \;

Vim

vim实用命令

  • 一般模式
命令 含义
w 跳到下一个单词的头部
⇧ + w 跳到下一个单词头部(含标点)
e 跳到下个单词的尾部
⇧ + e 跳到下个单词的尾部(含标点)
b 跳到上个单词头部
⇧ + b 跳到上个单词头部(含标点)
⇧ + h 跳到窗口第一行
⇧ + m 跳到窗口中间行
⇧ + l 跳到窗口最后一行
{ 向下，按块跳
} 向上，按块跳
] + m 跳到下一个大括号的 {
[ + m 跳到上一个大括号的 {
  • 编辑模式
命令 含义
i 光标所在字符进入插入模式
⇧ + i 跳到行首进入插入模式
a 光标往后移一位，进入插入模式
⇧ + a 跳到行尾，进入插入模式
o 当前行下一行新建空行，并进入插入模式
  • 视图模式
命令 含义
v 进入视图模式(相当于鼠标左键按下并选中某一部分)
  • 删除
命令 含义
x 剪切，相当于command + x
s 删除光标所在字符，并进入插入模式
cc 清空当前行，并进入插入模式
dd 删除当前行，但不进入插入模式
di + ( 删除小括号内的内容
di + { 删除大括号内的内容
: a, b d 删除 a 到 b 行的内容
ndd 从当前行删除n行
  • 命令行模式
  • s：substitute 替换
  • g：global全局
  • c：confirm确认
命令 含义
/pattern 从光标位置向下查找，按下n表示下一个，N表示上一个
?pattern 从光标位置向上查找，按下n表示下一个，N表示上一个
:%s/p1/p2/g 将文中所有p1替换成p2
:%s/p1/p2/g 将文中所有p1替换成p2，替换时需要确认
