基本操作

合成

  1. 双排管抽真空
  2. 减压蒸馏
  3. 常压蒸馏
  4. 萃取
  5. 重结晶
  6. 点板
  7. 过柱

聚合

  1. 乙烯瓶操作
  2. MAO取用

配碱缸

  1. 使用乙醇与氢氧化钠
  2. 将10L左右乙醇倒进碱缸
  3. 烧杯中到150ml左右乙醇，挖取氢氧化钠搅拌溶解至饱和溶液，溶解后倒进碱缸
  4. 重复步骤三3次

真空系统(schlenk系统)

  1. 由双排管，干燥塔和油泵，组成一个真空系统
  2. 双排管下面是真空通路，上面是氮气通路，通过旋转二通旋塞，切换抽真空和通氮气操作
  3. 使用时，先把要抽真空的瓶子接到双排管上，然后打开真空泵，氮气总阀
  4. 先旋转二通旋塞，接双排管的真空通路，抽真空，看水银柱液压，抽的时候在酒精灯上加热排出水分
  5. 然后旋转二通旋塞，停掉抽真空，接通双排管氮气通路，充氮气
  6. 重复步骤4、5，反复3次得到充满氮气的无水无氧环境
  7. 停掉真空泵，先旋转其他二通旋塞，使双排管通大气，然后再关闭电机，防止泵油倒吸进双排管
  8. 在通氮气环境下，加反应物进行反应

溶剂极性

常见的溶剂极性大小顺序（由小至大）为：石油醚、环己烷、四氯化碳、苯、甲苯、二氯乙烷、二氯甲烷、三氯乙烯、二苯醚、氯仿、正丁醚、乙醚、DME、硝基苯、二氧六环、三辛胺、四氢呋喃、乙酸乙酯、三丁胺、甲酸甲酯、三乙胺、丙酮、苯甲醇、吡啶、正丁醇、异丙醇、乙二醇、乙醇、乙酸、甘油（丙三醇）、乙腈、DMF、甲醇、六甲基磷酰胺、甲酸、DMSO、三氟乙酸、甲酰胺、水、三氟甲磺酸、无水硫酸、无水高氯酸、无水氢氟酸。

其中三氟乙酸，三氟甲磺酸，无水硫酸、无水高氯酸、无水氢氟酸等强酸由于腐蚀性极强，实际上在一般实验中应用不多，这里只是列出以便比较物质极性大小而已

通常柱色谱常用有机溶剂为石油醚、环己烷、二氯甲烷、三氯乙烯、乙醚、DME、二氧六环、四氢呋喃、乙酸乙酯、丙酮、乙醇、乙酸、甲醇这几种溶剂，至于具体问题，则经常使用几种溶剂的混合溶剂来进行分离物质

蒸甲苯

  1. 压钠丝，放到盛有甲苯的大圆底烧瓶中，放二苯甲酮做显色剂
  2. 圆底烧瓶接油水分离器，接接收器
  3. 接冷凝管，通水
  4. 加热回流4h后，关闭油水分离器，开始收集
  5. 使用时使用抽烤过的针头，从接收器中抽取甲苯

溴化镍除水（NiBr2·4H2O）

  1. 蒸甲苯备用
  2. 称取四水合溴化镍，放于史莱克瓶中，加入甲苯没过溴化镍固体，放入磁子
  3. 史莱克瓶加油水分离器，接冷凝管，油浴加热，120℃回流除水
  4. 待水液面不再升高，停止除水
  5. 使用冷肼抽干史莱克，得黄色固体，即除水后的溴化镍

减压蒸馏

  1. 圆底烧瓶加待蒸馏液体，之前是减压蒸馏苯甲酰氯作原料，温度118-120度，液体不多于1/2，不少于1/3

  2. 烧瓶接蒸馏头

  3. 温度计插到蒸馏头上面，水银球上沿和蒸馏头支管下沿保持水平

  4. 冷凝管通水，接蒸馏头支管

  5. 使用圆底烧瓶接收，有裂缝的不要用，防止爆掉

  6. 油浴加热圆底烧瓶，液面超过2/3，用抹布包裹蒸馏头加快速度

  7. 开启搅拌机

  8. 打开油泵，双排管二通旋塞调节真空度

  9. 调节到减压温度，开始加热，前馏分会早蒸出来，等升到减压温度稳定下来，用另外的圆底瓶接收较纯馏分，这里就是苯甲酰氯

重结晶

  1. 使用布氏漏斗干燥，除掉固体杂质
  2. 使用溶剂溶解固体，溶剂常用的乙酸乙酯，二氯甲烷
  3. 放冰箱得到结晶固体

过柱子(管柱层析)

  1. 先用一个烧杯盛放矽胶，也就是要用到的填充物
  2. 再用另一个烧杯盛放冲提剂，搅拌加入上一步的矽胶，搅拌到稀泥状，没有沉淀的时候停下
  3. 把少量棉花放到柱子底端
  4. 铁架台夹住柱子，下面放一干净烧杯接收冲提剂
  5. 往柱子里加冲提剂冲一下底部棉花，然后用玻璃棒把棉花中气泡压出来
  6. 棉花上铺一层无水硫酸钠，使用冲提剂把粘在管壁上的冲下去
  7. 然后把早准备好的填充物倒进柱子，使用冲提剂把烧杯里残留的矽胶，和柱子上沾着的矽胶冲下去
  8. 等待沉降，这个过程中可以不停弹柱子让填充物压实，压实是为了保持填充物面平整，保证更好的分离效果
  9. 等冲提剂下降到填充剂表面的时候，可以添加些无水硫酸钠做缓冲，也是为了避免后面倒分离液体的时候破坏平整的柱面，保证分离效果
  10. 等冲提剂下降到基本接近填充剂表面的时候，使用胶头滴管抽要分离的液体，靠近填充剂，沿着管壁环形加入液体
  11. 用另一个空烧杯，换下下面之前放置的烧杯
  12. 之前放置的烧杯里面都是干净的冲提剂，这时候把冲提剂导入柱子，洗第11步加入的待分离液体即可
  13. 空烧杯中滴下的液体每隔一段时间点板，根据点板情况收集自己的产物即可
  14. 要注意在冲提的过程中不能让冲提剂低于矽胶上沿，不停加冲提剂即可
  15. 结束后把杂质倒进废液桶，把柱子倒过来滴下水，把填充物倒出来洗洗干净

萃取

  1. 在萃取前，使用萃取液检查是否漏液

  2. 将梨形分液漏斗放到铁架台上，下方放置空烧杯

  3. 先加入萃取溶剂，再加入待萃取溶液

  4. 梨形分液漏斗内的液体不要超过3/4

  5. 将瓶口塞住，不要旋转

  6. 拿下分液漏斗，顶住瓶塞，倒置，旋转活塞放一次气，然后关闭活塞，晃动，差不多晃动5次打开活塞放气

  7. 将分液漏斗放到铁架台上，取下瓶塞，静置一会，产生分层

  8. 旋转活塞，收集下层溶液，剩下的上层溶液从上口倒出

  9. 如果有水，加入少量无水硫酸镁除水，然后过滤到另外的干净瓶子中

  10. 萃取完成，蒸馏或者减压蒸馏就可以得到干净的萃取物

席夫碱反应

骨架和成主要就是席夫碱反应，使用等物质的量的醛和胺缩合形成席夫碱

芳香胺与羰基化合物发生亲核加成，生成类似半缩醛的半胺醛，然后发生失水反应生成亚胺

MAO取用

  1. 门后面放着甲基铝氧烷的甲苯溶液(MAO)，用来做聚合反应的助催化剂
  2. 准备羊角瓶，耐腐蚀乳胶管3根，导管1根，钢丝4节
  1. 取用前摇晃MAO缸，因为MAO比甲苯密度大，在下面，如果量少的话可能压不上来
  2. 按照上图接好装置导管一侧朝向玻璃棒方向插进乳胶管，目的是为了结束取用的时候把乳胶管内的MAO赶到羊角瓶中，另一侧最好插进羊角瓶的长角，刚开始不要插太深，防止抽烤时把导管烤化
  3. 抽烤羊角瓶3次，使其充满氩气，待羊角瓶冷却后，将导管从长角中深入羊角瓶
  4. 接双排管一侧的乳胶管插针头，否则因为压力会压不出来，逆时针旋转开关，导出MAO到羊角瓶
  5. 导出完成后，关闭开关，赶出乳胶管内的MAO
  6. 拔出导管，用长角堵住乳胶管和导管接触的孔，取用时就从这个小孔取
  7. 短脚一侧取下针头，用夹子夹住乳胶管，接玻璃棒堵住针头孔

乙烯均聚

  1. 使用合出的络合物，利用金属原子的空位，催化乙烯等聚合
  2. 聚合前接好装置，不要忘记聚合釜中加搅拌子，使用油浴加热，抽烤反应釜
  3. 冷却，使用乙烯与Ar气置换3次
  4. 在通乙烯保护条件下，加入催化剂体系和溶剂，如果与MMA共聚，在这一步加入MMA
  5. 通乙烯，待压力稳定后加入MAO
  6. 继续通乙烯至所需压力，聚合
  7. 聚合一段时间后，使用5%的盐酸+乙醇溶液终止反应
  8. 过滤聚合物，用无水乙醇洗涤，真空干燥至恒重，称量

MMA聚合

  1. MMA的聚合在干燥，洁净的安培瓶中进行
  2. 在氩气保护下，在安培瓶中依次加入催化剂，溶剂，MMA，再加入MAO
  3. 将反应物混合均匀，置于恒温水浴中加热，反应一段时间
  4. 打开安培瓶，将反应物倒入体积分数5%的HCl的乙醇溶液中终止聚合
  5. 聚合物过滤，洗涤，真空干燥，称重

文献1

合成

2个配体，4个中心原子，2个前过渡，2个后过渡，合成8种催化剂

聚合

筛选出能在单一釜中催化乙烯/丙烯共聚合得到PE-b-P(E/P)多嵌段共聚物的催化剂

聚合步骤：

  1. 抽烤双支管 300mL 史莱克瓶，与单支管300mL史莱克瓶
  2. 通Ar气环境下，双支管史莱克加入催化剂
  3. 通Ar气环境下，单支管史莱克加干燥过的甲苯80mL，与定量MAO，搅拌15分钟
  4. 通Ar气环境下，通过四氟乙烯导管将MAO加到催化剂所在的史莱克瓶
  5. 乙烯与丙烯置换双支口史莱克，反应1小时，通过控制乙烯丙烯进料压力控制投料比
  6. 聚合结束后用10%的盐酸/乙醇溶液终止反应
  7. 产物过滤，酒精与蒸馏水洗涤，60℃恒温箱烘干，称重

结果

考虑 中心原子 陈化条件(聚合温度，AI/Ni比，催化剂浓度，乙烯与丙烯投料比，聚合时间)对催化剂催化乙烯丙烯共聚的影响

检测

红外检测，检测是否有聚丙烯长链段

核磁碳谱，计算不同碳原子化学位移，计算竞聚率

综合红外与核磁，得出无规共聚的结论

DSC分析

a 聚乙烯均聚物

b 丙烯含量2.6mol%的乙丙共聚物

c 丙烯含量5.1mol%的乙丙共聚物

d 丙烯含量11.5mol%的乙丙共聚物

对比4种DSC曲线

随着丙烯含量增加，聚乙烯长链段显示的熔点逐渐降低，当丙烯含量达到11.5mol%时，共聚物熔点不明显，说明插入成功

收获

在做配体的时候，如果骨架有吸电子基，有利于降低中心原子的电子云密度，利于共聚单体插入

后过渡金属的空间位阻大，聚合单体浓度低或者催化位阻较大的长链α-烯烃效果更好

文献2

PMMA嵌段可以显著降低结晶与熔融热

之所以做这个嵌段共聚，是为了克服聚乙烯不易与其他材料粘合的缺陷，可以用来作相容剂，提高可染性，说白了就是用在染色剂方向

PE-b-stereoPMMA二嵌段共聚物，PE做硬嵌段，PMMA做软嵌段，能让PE结晶温度降低

功能软嵌段对PE结晶动力学影响很有趣，但是之前也没人做过，所以有了这篇文章

MMA与烯烃顺序共聚通常使用茂金属，生成的聚烯烃短链少支链也少，而且茂金属可以极大地改变PMMA嵌段的立体规则性，之前实验室于栋平师姐好像是做的茂金属催化剂

收获

如果按照这篇文献来做，一个要做催化剂催化乙烯均聚，一个要做催化剂催化MMA均聚，这两组实验作对照，另外一个做共聚

MMA均聚

  1. MMA纯化，甲苯干燥
  2. 抽烤250mL三口瓶
  3. 加入15mL甲苯，0.1mL 三异丁基铝 搅拌10分钟
  4. 4μmol催化剂溶在4mL干燥的甲苯溶液中，加入三口瓶
  5. 添加 0.1mL MAO
  6. 10分钟后，在通Ar气环境下，添加3mL纯化的MMA
  7. 室温反应90分钟
  8. 使用酸性甲醇淬灭反应，使用过量甲醇沉淀得到PMMA
  9. 甲醇洗涤3次，干燥，过夜，称重，产量2.650g

乙烯均聚

  1. 乙烯高压釜抽烤，甲苯干燥

  2. 15mL甲苯加入到聚合釜

  3. 4μmol催化剂溶在4mL干燥的甲苯溶液中，加入到聚合釜

  4. 0.1mL MAO加入到聚合釜

  5. 在2个大气压下通乙烯，聚合15分钟

  6. 淬灭，产物分离，洗涤，干燥，过夜，称重，产量0.350g

乙烯与MMA嵌段共聚

  1. 乙烯高压釜抽烤，甲苯干燥
  2. 15mL甲苯加入到聚合釜
  3. 4μmol催化剂溶在4mL干燥的甲苯溶液中，加入到聚合釜
  4. 0.1mL MAO加入到聚合釜
  5. 在2个大气压下通乙烯，聚合15分钟
  6. 乙烯进料15分钟后，加入MMA，继续聚合45分钟
  7. 淬灭，产物分离，洗涤，干燥，过夜，称重，产量0.500 g
